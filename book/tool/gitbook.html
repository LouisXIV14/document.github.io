gitbook

安装卸载

sudo npm install gitbook-cli -g # 安装
gitbook --version # 查看版本信息
npm update gitbook-cli -g # 查看版本
sudo npm uninstall gitbook-cli -g # 卸载

初始化

gitbook init # 只支持生成两级目录
# warn: no summary file in this book
# info: create README.md
# info: create SUMMARY.md
# info: initialization is finished
  • README.md: 项目的介绍
  • SUMMARY.md: GitBook 的目录结构,用于生成html文件

SUMMARY.md

# Summary

启动服务

gitbook serve # http://127.0.0.1:4000/

优缺点

  • GitBook最大的问题在于当文件很多时，每次编辑后的重新加载时间长得令人无法忍受
  • 直接加载markdown，不能直接加载html
  • 它的默认主题导航结构也比较有限制性
  • 主题系统也不是 Vue 驱动的

  • GitBook 团队如今也更专注于将其打造为一个商业产品而不是开源工具

  • 目录及文件名可以是中文的
