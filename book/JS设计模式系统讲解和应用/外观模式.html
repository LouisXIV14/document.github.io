外观模式

外观模式介绍

  • 为子系统中的一组接口提供了一个高层接口
  • 使用者使用这个高层接口

UML类图

场景

function addEvent(el, type, selector, fn) {
    if (fn == null) {
        fn = selector
        selector = null
    }
    // ----
}
// 调用时可以传入两种参数
addEvent(el, 'click', '#div1', fn)
addEvent(el, 'click', fn)

设计原则验证

  • 不符合单一职责原则和开放封闭原则，因此谨慎使用不可滥用
